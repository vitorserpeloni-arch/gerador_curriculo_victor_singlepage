# Gerador de Currículos - Victor Serpeloni (Single Page)

Projeto em página única que gera o currículo na mesma tela. Minimalista, feito com Tailwind CSS.

## Como usar localmente (XAMPP)

1. Copie a pasta `gerador_curriculo_victor_singlepage` para o diretório `htdocs` do XAMPP (ex: `C:\xampp\htdocs\gerador_curriculo_victor_singlepage`).
2. Inicie o Apache no XAMPP Control Panel.
3. Abra o navegador e acesse: `http://localhost/gerador_curriculo_victor_singlepage/index.php`

## Características
- Geração do currículo **na mesma página** sem envio para backend.
- Impressão / download via `window.print()`.
- Adição e remoção dinâmica de experiências.
- Layout minimalista com Tailwind CSS.
